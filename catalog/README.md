# DockJob TrueNAS Catalog Assets

This directory contains assets for the TrueNAS custom app catalog entry.

## Logo Requirements

For TrueNAS custom apps, the logo should be:
- Format: PNG, SVG, or JPG
- Size: Recommended 256x256px or 512x512px
- Transparent background (PNG) preferred
- Square aspect ratio

## Usage

1. Place your logo file in this directory (e.g., `dockjob-logo.png`)
2. Upload to a publicly accessible URL (GitHub raw, etc.)
3. Reference the URL in your TrueNAS app catalog configuration

## Example URLs

If you place a logo here, you can reference it via GitHub raw URL:
```
https://raw.githubusercontent.com/matepalocska/dockjob/main/catalog/dockjob-logo.png
```
